Desafío Día 1

  1. Se declararon las variables:
    numeroUno = 1
    stringUno = '1'
    numeroTreinta = 30
    stringTreinta = '30'
    numeroDiez = 10
    stringDiez = '10';
  2. Hacer Click en iniciar.
  3. Observar el mensaje con la comparación
  4. Por coerción de tipos "numeroDiez == stringDiez" es "true"
    "numeroDiez !== stringDiez" compara tipos diferentes también es "true"
    . (true && true) → true se ejecuta el if
  5. El triple igual (===) compara valor y tipo al mismo tiempo "numeroTreinta === stringTreinta" es "false"
  6. Nota: !== (negación estricta) es lo opuesto de === (comparación estricta) evalúa como true si el valor o tipo es diferente
