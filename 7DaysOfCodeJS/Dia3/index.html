Desafío Día 1

  1. Hacer Click en iniciar.
  2. Por coerción de tipos "numeroDiez == stringDiez" es "true"
    "numeroDiez !== stringDiez" compara tipos diferentes es "true"
    Por lo tanto (true && true) → true se ejecuta el if
  3. El triple igual (===) compara valor y tipo al mismo tiempo "numeroTreinta === stringTreinta" es "false"
  4. Nota: !== (negación estricta) es lo opuesto de === (comparación estricta) evalúa como true si el valor o tipo es diferente
  5. Variables declaradas :
    [numeroUno = 1] [stringUno = '1']
    [numeroTreinta = 30] [stringTreinta = '30']
    [numeroDiez = 10] [stringDiez = '10']

Desafío Día 2

  • Al abrir esta página, se te pedirá que ingreses tu nombre.
  • Luego, deberás ingresar tu edad. Solo se aceptan números mayores que 0.
  • Después, el sistema te preguntará qué lenguaje de programación estás estudiando.
  • Finalmente, deberás responder si te gusta estudiar ese lenguaje, digita:
    • 1 para SÍ
    • 2 para NO
  • Si ingresas datos inválidos (vacíos, letras donde van números, etc.), el sistema te volverá a preguntar hasta que ingreses algo correcto.
