Desafío Día 2

¡Hola! Este archivo HTML sirve como contenedor para cargar el script JS que hará preguntas al usuario usando prompt() y mostrará mensajes con alert().

  • Al abrir esta página, se te pedirá que ingreses tu nombre.
  • Luego, deberás ingresar tu edad. Solo se aceptan números mayores que 0.
  • Después, el sistema te preguntará qué lenguaje de programación estás estudiando.
  • Finalmente, deberás responder si te gusta estudiar ese lenguaje, digita:
    • 1 para SÍ
    • 2 para NO
  • Si ingresas datos inválidos (vacíos, letras donde van números, etc.), el sistema te volverá a preguntar hasta que ingreses algo correcto.
